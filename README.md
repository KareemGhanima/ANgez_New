# Angez - Gamified Productivity App

Welcome to Angez! This app turns your daily routines into an RPG experience.

## Installation

As Node.js was not detected on your system during scaffolding, the codebase has been hand-crafted using Next.js App Router rules. 

To run this on your machine, follow these steps:
1. Install [Node.js](https://nodejs.org/)
2. Open this directory (`scratch/angez`) in your terminal.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open [http://localhost:3000](http://localhost:3000)

## Features Included
- **Customizable Dark Gamified UI**: Select "Primary Focus" and "Action Accent" colors via the gear icon.
- **Quest System**:
  - `Complete`: Gain XP.
  - `Interested`: Gives a thumbs-up, telling the recommender to show more.
  - `Later`: Sets the tag to later.
  - `Skip`: Skips the quest and reduces the interested score.
- **Dynamic Avatar Viewer**: Changes appearance when your XP significantly grows.
- **User Progression**: Tracks exact XP and displays Level and attributes.

Enjoy leveling up!
